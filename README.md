# Student Grading System

## Description
This is a simple Python-based Student Grading System that allows users to:

- Enter student names
- Validate student names
- Enter student IDs
- Validate student IDs
- Input student scores
- Automatically calculate grades
- Grade multiple students continuously

The program uses loops, conditional statements, and input validation to ensure correct data entry.

---

## Features
- Name validation
- Student ID validation
- Grade calculation
- Continuous grading system
- User-friendly interface

---

## Grading Criteria

| Score Range | Grade |
|------------|-------|
| 90 - 100  | A |
| 80 - 89   | B |
| 70 - 79   | C |
| 60 - 69   | D |
| Below 60  | F |
